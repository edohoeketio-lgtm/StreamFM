# StreamFM

An interactive DJ simulator and custom audio engine built with React and the native browser Web Audio API. Mix tracks, apply real-time effects, and perform live sets entirely in the browser.

## Features

- Real-time audio mixing with crossfader
- Custom Web Audio API synthesis engine
- FX rack with filters, reverb, and delay
- Waveform visualization
- Track library management

## Tech Stack

- React
- Web Audio API
- Canvas API (waveforms)
- JavaScript (ES6+)

## Running Locally

```
npm install
npm run dev
```
